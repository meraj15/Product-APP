# Product-App

This is a Flutter project for a e-commerce products application. <br>

1. **Clone the Repository**
   ```sh
   git clone https://github.com/meraj15/Product-App.git

2. **Navigate to the Project Directory**
   ```sh
   cd Product-App

3. **Get Flutter Dependencies**
   ```sh
   flutter pub get

4. **Run the App**
   ```sh
   flutter run
